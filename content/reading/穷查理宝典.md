# 穷查理宝典

《穷查理宝典》（Poor Charlie's Almanack）是查理·芒格（Charlie Munger，1924—2023）的思想文集，由彼得·考夫曼编辑，2005年首版。全书收录了芒格多年间的演讲与备忘录，核心主张是：**用多个学科的思维模型构建认知框架，比单一领域的专业知识更能看清世界**。

## 多元思维模型

芒格认为，大多数人只用本专业的框架理解问题，这会导致系统性盲点——"在手里只有锤子的人眼中，一切问题都像钉子"。他建议从以下学科中各提取最重要的模型：

- **物理学**：临界质量、系统转变
- **生物学**：自然选择、适应
- **心理学**：25种认知偏误（详见下方）
- **经济学**：机会成本、激励扭曲
- **数学**：概率、复利、排列组合

## 心理偏误清单

芒格的"人类误判心理学"演讲列举了25种认知偏误，影响最广的包括：

| 偏误 | 描述 |
|------|------|
| 激励扭曲 | 人会按照激励行事，而非按道德行事——设计制度必须考虑激励结构 |
| 社会认同 | 在不确定时跟随多数人，导致群体性错误 |
| 喜爱/讨好偏误 | 对喜欢的人的请求更难拒绝，对其缺点视而不见 |
| 厌恶损失 | 损失的痛苦是同等收益快乐的2倍，导致持亏盘不割 |
| 禀赋效应 | 拥有某物后，评估其价值高于市场价 |
| 沉没成本 | 因已投入大量资源而继续追加，而非根据未来预期做决策 |

## 与价值投资的关系

芒格是[[巴菲特]]的长期合伙人，他将心理学和跨学科思维引入伯克希尔的投资决策，补充了巴菲特偏定量的格雷厄姆体系。[[段永平]]深受芒格影响，"本分"框架中对人性、激励与企业文化的重视，与芒格的心理偏误分析高度呼应。
